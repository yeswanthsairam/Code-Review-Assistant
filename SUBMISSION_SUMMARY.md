This prototype tackles Challenge 3: Code Review Assistant. I wrote a production-ready prompt template that asks an LLM to review short snippets with a strict output format: exactly three actionable improvements and one positive note. To make it practical, I added a tiny Python utility that loads sample snippets and generates a final prompt you can paste into ChatGPT or Gemini. The examples use only self-created dummy code, so no confidential data is involved. The design emphasizes readability, maintainability, and developer-friendly tone. The result is lightweight, explainable, and easy to demo using screenshots or a shared gist link for review.
